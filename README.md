# Bot Setup Instructions

## Prerequisites
- Make sure you have Python 3.x installed on your machine.
- Install the required packages by running:
    ```bash
    pip install -r requirements.txt
    ```

## Running the Bot
1. Clone the repository:
    ```bash
    git clone https://github.com/mariusz123334434/bot.git
    cd bot
    ```
2. Start the bot using:
    ```bash
    python bot.py
    ```

## Configuration
- Update the `config.json` file with your specific settings before running the bot.
